Infrastructure

fDeploy Server infrastructure overview displays the following key areas:

  • Deployment machine (also known as target) status
  • Deployment machine roles (also known as target roles)
  • Environments

Target status

The target status card provides a summary of all deployment target health states:

  • Healthy — targets that are online and responding to health checks
  • Unavailable — targets that are not responding
  • Disabled — targets that have been manually disabled

Clicking on a status category navigates to the deployment targets list filtered by that status.

Target roles

The target roles card displays all unique roles assigned across your deployment targets. Clicking on a role navigates to the deployment targets list filtered by that role.

Environments

The environments card lists all configured environments. Clicking on an environment navigates to its detail page.
