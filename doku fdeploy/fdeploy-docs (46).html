Session settings

The session settings page allows administrators to configure how user sessions behave, including timeout duration and sliding expiration.

Note

Session settings are only accessible to users with the System Administrator role.

Accessing session settings

Navigate to Configuration → Session to manage session configuration.

Session timeout

The Session Timeout field specifies how long a user session remains valid, in minutes. A human-readable duration is displayed below the input (e.g., “2 hours 30 minutes”) to help you verify the configured value.

Sliding expiration

When Sliding Expiration is enabled, the session timeout is reset each time the user performs an action. This means users who remain active will not be signed out.

When disabled, the session expires after the configured timeout regardless of user activity.

Saving changes

After modifying the settings, click Save to apply the changes.
